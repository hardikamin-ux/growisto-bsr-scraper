Get started
Install in under 2 minutes
One command on Mac, one double-click on Windows. Everything is set up automatically including the desktop shortcut.
🍎
Mac
macOS 11+ · Python 3.9+
Run this once in Terminal
curl -fsSL https://raw.githubusercontent.com/hardikamin-ux/growisto-bsr-scraper/main/install_mac.sh | bash
  • Downloads all files to ~/growisto-bsr-scraper/
  • Installs Python packages automatically
  • Installs Chromium browser for scraping
  • Creates "Growisto BSR Scraper" shortcut on Desktop
🪟
Windows
Windows 10/11 · Python 3.9+
Step 1 — Download the installer
⬇️ Download install_windows.bat
  • Double-click the downloaded .bat file
  • Downloads all files to ~/growisto-bsr-scraper/
  • Installs Python packages and Chromium automatically
  • Creates "Growisto BSR Scraper" shortcut on Desktop

App password (share with your team)
