Get started
Download & Install
One zip file. Run the installer once. A desktop shortcut is created automatically.
🍎
Mac
macOS 11+ · Python 3.9+ required
⬇️ Download for Mac
  • Download and unzip the file
  • Open Terminal, drag install_mac.sh into it, press Enter
  • Installs all packages + Chromium (~3–5 min)
  • Growisto BSR Scraper shortcut appears on your Desktop
  • Double-click the shortcut anytime to launch
If macOS blocks the file: System Settings → Privacy & Security → Open Anyway
🪟
Windows
Windows 10/11 · Python 3.11 or 3.12 required
⬇️ Download for Windows
  • Install Python 3.12 first (see below) — required
  • Download and unzip the BSR file above
  • Right-click install_windows.bat → Run as administrator
  • Installs all packages + Chromium (~3–5 min)
  • Growisto BSR Scraper shortcut appears on your Desktop
  • Double-click the shortcut anytime to launch
If Windows Defender warns: click More info → Run anyway

⚠️ PYTHON SETUP — WINDOWS USERS (DO THIS FIRST)

Important: Use Python 3.11 or 3.12 only. The latest version (3.14) is too new and breaks Streamlit. Do NOT use Python from the Microsoft Store.

  1. Go to python.org/downloads/release/python-31210 (Python 3.12.10)
  2. Scroll down to the "Files" section
  3. Click "Windows installer (64-bit)" to download
  4. Open the downloaded .exe file
  5. CRITICAL: On the first screen, tick the box at the bottom — "Add python.exe to PATH"
  6. Click "Install Now" and wait ~2 minutes
  7. Restart your computer so PATH takes effect
  8. Now run the BSR installer above

Already have Python 3.14 or Microsoft Store Python? Uninstall it first: Settings → Apps → search for Python → Uninstall. Then follow the steps above.

Setup guide
How to set up
One-time setup takes about 5 minutes.
1
Download & unzip
Click Download for Mac above. Find the zip in your Downloads folder and double-click to unzip it.
2
Run the installer
Open Terminal. Drag install_mac.sh from the unzipped folder into Terminal, then press Enter.
bash ~/Downloads/growisto-bsr-scraper-main/install_mac.sh
3
Wait for setup
The installer downloads Python packages and Chromium (~3–5 minutes). Don't close the Terminal window.
4
Launch the app
A Growisto BSR Scraper shortcut appears on your Desktop. Double-click it — the app opens in your browser automatically.
1
Download & unzip
Click Download for Windows above. Right-click the zip → Extract All → choose a folder.
2
Run the installer
Inside the unzipped folder, right-click install_windows.bat → Run as administrator. If Windows Defender warns: click More info → Run anyway.
3
Wait for setup
The installer downloads Python packages and Chromium (~3–5 minutes). Don't close the Command Prompt window.
4
Launch the app
A Growisto BSR Scraper shortcut appears on your Desktop. Double-click it — the app opens in your browser automatically.
